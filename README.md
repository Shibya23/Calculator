A calculator implemented on the DEO-CV FPGA board 

Can do addition/subtraction on numbers 0-15 and outputs results from 0-30

Far left switches are meant to input number in binary while push buttons zero and one serve as an enter key to input operands

Results are displayed on the 7-Segment display 

This calculator does not handle negatives well and will usually output the wrong answer

The .csv file contains the original pin planner of this project
